# C++ 个人实践仓库 🚀

> 大一 C++ 程序设计课程学习记录与代码归档

## 📖 关于本仓库

本仓库主要用于记录我在大一下学期学习《C++ 程序设计》课程的完整历程。这里不仅保存了从基础语法到面向对象编程的完整练习代码，还整理了编译调试技巧、常见错误分析及重构优化思路。

仓库将持续更新，见证我从入门到熟练掌握 C++ 的每一步积累。

## 📂 目录结构

| 文件夹 | 内容说明 |

## 🛠️ 涉及知识点

- **基础语法**: 变量、控制流、数组与指针
- **面向对象**: 类与对象、继承、多态、封装
- **STL 标准库**: Vector, String, Map 等容器的使用
- **内存管理**: 动态内存分配与释放
- **文件操作**: 输入输出流处理

## 📝 学习心得

在编写这些代码的过程中，我深刻体会到了 C++ 严谨性与灵活性的并存。通过不断的 Debug 和重构，我对底层内存模型有了更直观的理解。
